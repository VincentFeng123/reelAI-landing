# Three.js Hero Scene Skill

When building 3D landing page scenes:

## Scene Rules
- Use Three.js or React Three Fiber.
- Keep the scene minimal but premium.
- Use one strong hero object, not clutter.
- Add subtle floating motion.
- Add mouse parallax.
- Tie camera position, rotation, or object transforms to scroll.

## Lighting
- Use soft studio lighting.
- Use ambient light + directional/spot lighting.
- Avoid harsh overexposed colors.
- Prefer black, white, chrome, glass, and subtle glow.

## Performance
- Use low-poly or optimized geometry.
- Compress textures.
- Lazy-load the 3D scene.
- Pause or simplify animation on mobile.
- Respect `prefers-reduced-motion`.

## ReelAI Style
For ReelAI, the best hero object is:
- an infinite ribbon of floating reel cards
- a neural media core
- a portal of endless content
- a floating phone exploding into video cards

The primary recommendation is an infinite ribbon of video cards forming a scroll tunnel.
