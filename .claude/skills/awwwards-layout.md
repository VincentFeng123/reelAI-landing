# Awwwards Layout Skill

Avoid generic AI landing page layouts.

## Do Not Do
- Do not make every section centered.
- Do not use repetitive card grids.
- Do not use stock SaaS layouts.
- Do not use boring “feature card, feature card, feature card” sections.

## Do Instead
- Use asymmetrical layouts.
- Use oversized typography.
- Use large negative space.
- Use layered elements.
- Let sections feel like scenes.
- Make every scroll section visually different.

## Layout Patterns
- Split-screen text + 3D object.
- Full-screen pinned cinematic scene.
- Horizontal scroll gallery.
- Floating cards around a 3D core.
- Timeline reveal.
- Infinite tunnel.
- Diagonal section transitions.
